如何恢复记账数据

迁移记账数据

1. iCloud 数据恢复

如果开启了 iCloud 自动备份，在用户使用过程中，每隔一个小时，都会尝试将数据上传到 iCloud 空间，所以更换设备之后，在新的设备登陆原有设备的 AppleID，正常开启 iCloud 功能，然后手动恢复最新的数据即可。

手动备份与恢复数据的方式如下：

  • 在旧设备 app 设置 -> 数据备份 -> 选择 备份数据；
  • 待备份成功后，在新设备 app 设置 -> 数据备份 -> 选择 恢复数据。

需要注意的是，备份数据和恢复数据都需要保持网络畅通，iCloud 功能正常开启。

2. 数据库恢复

在旧的设备，可以在“设置”-》“数据库操作”中导出一份数据库，并发送到另外一台设备。并在这个新的设备上，执行“设置”-〉“数据库恢复”即可。
