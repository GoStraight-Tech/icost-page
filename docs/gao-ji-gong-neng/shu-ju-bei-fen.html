数据备份与恢复

数据备份与恢复

iCost 的备份功能是将用户的记账数据直接保存到每个用户的 iCloud 空间下（用户独享的空间，依赖由苹果维护的云盘系统），而我们则不对这部分数据进行任何读取或者保存。

1. 如果更换手机，是否会遗失数据？

数据不会遗失。如果开启了 iCloud 自动备份，在用户使用过程中，每隔一个小时，都会尝试将数据上传到 iCloud 空间，所以更换设备之后，在新的设备登陆原有设备的 AppleID，正常开启 iCloud 功能，数据就会进行恢复。

如果新设备上未自动恢复成功旧设备的数据，可以采用以下方式进行手动恢复：

  • 在旧设备 app 设置 -> 数据备份 -> 选择“备份数据“；
  • 待备份成功后，在新设备 app 设置 -> 数据备份 -> 选择"恢复数据"。

需要注意的是，备份数据和恢复数据都需要保持网络畅通，iCloud 功能正常开启。

2. iCloud 备份功能，是否会泄露隐私？

不会，隐私问题是我们非常看重的。

事实上，在处理 iCloud 的数据问题上，我们付出了非常多的精力，而这都是因为 iCloud 备份功能有一个非常大的优点，就是用户隐私可以得到充分的保障，iCloud 功能是由苹果公司提供且维护的，数据存储在每个用户的独立云盘空间内，不与其他用户的数据存放在一起。

另外，苹果公司在数据网络传输上有着非常严格的校验机制，传统的攻击手段例如中间人攻击是无法破坏数据的完整性的，我们有理由相信 iCloud 机制下数据传输的安全性。

3. iCloud 数据的传输是安全的，意味着数据 100% 不会出问题吗？

在以下的情况，数据自动备份会存在不稳定的情况：

  • 您所处的网络环境不稳定，此时数据无法稳定传输到远程，需要等待网络环境好转，才能进行完整备份。

由于涉及网络的请求，本身具有失败的可能，另外受限网络环境的影响，可能出现数据无法及时备份的情况，不过不需要担心，如果数据无法及时备份，请采取以下措施尝试解决：

  • 检测设备网络状态，尝试切换成另一个网络，重新尝试。

我们也会持续不断的优化 iCloud 数据备份功能，尽力保护您的数据安全。

4. iCloud 自动备份功能如何开启和关闭？

4.1 开启 iCloud 自动备份功能

需要检查开关：

  • 应用内部的设置：iCost 应用内部的设置页面 -> iCloud 备份；

  • 系统的 iCloud 配置：

    系统设置页面 -> Apple ID、iCloud、媒体与购买项目 -> iCloud -> 使用 ICLOUD 的 APP -> iCost；

    系统设置页面 -> Apple ID、iCloud、媒体与购买项目 -> iCloud -> iCloud 云盘；

4.2 关闭 iCloud 自动备份功能

如果想要关闭 iCloud 功能，只需要手动在应用设置里关闭备份的开关即可。

5. iCloud 备份失败可能的原因：

  • iCloud 存储空间不足；
  • 备份频率过高，需要等待一段时间后再次尝试。
